Summary: Lattitude vs. X

The aim of this project was to analyze the changes in weather with respect to the distance from the equator. Data was pulled from the OpenWeatherMap API to assemble data from over 500 cities. As you can see, there is a downward trend where the maximum temperature declines as we approach 80 degrees Lattitude. Between -40 and 20 degrees Lattitude, we see moderate temperatures, generally ranging from 50 - 100 degrees Farenheit; however, Lattitudes above 40 generally fall below 50 degrees Farenheit.
